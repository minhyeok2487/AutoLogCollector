Features

SecureCRT 같은 상용 프로그램 없이, 무료로 네트워크 장비를 관리하세요.

⚡

일괄 명령어 실행

다수의 Cisco 장비에 SSH 접속하여 명령어를 순차 실행합니다. 서버별 개별 인증도 지원합니다.

⏰

스케줄 자동 실행

일간, 주간, 월간 스케줄을 등록하여 자동으로 로그를 수집합니다. 스케줄별 로그 폴더 분리.

📊

Excel 자동 내보내기

실행 완료 후 결과를 Excel 파일로 자동 내보내기. 보고서 작성이 간편해집니다.

🔄

자동 업데이트

앱 내에서 최신 버전 확인 및 자동 업데이트. 항상 최신 기능을 사용할 수 있습니다.

📋

CSV/TXT Import/Export

서버 목록을 CSV로, 명령어를 TXT로 관리. 템플릿을 만들어 재사용할 수 있습니다.

🔒

서버별 개별 인증

글로벌 인증 외에 서버별 개별 SSH 자격 증명 설정. 다양한 환경에 유연하게 대응합니다.
